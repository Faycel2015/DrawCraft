## Description

Transform your ideas into reality with DrawCraft: Notes & Sketches – the professional-grade drawing and note-taking app designed for both iOS and macOS. Whether you're an artist, student, professional, or someone who loves to sketch, DrawCraft provides the perfect digital canvas for all your creative needs.

## Features

- 🎨 Multiple drawing tools (pen, pencil, marker, eraser)
- 🎯 PencilKit integration for pressure sensitivity
- 🌈 Rich color picker with customizable palettes
- ↩️ Undo/Redo functionality
- 💾 Save and load drawings
- 📱 Cross-platform support (iOS & macOS)
- 🎪 Haptic feedback for better user experience
- 📊 Layer support with opacity control
- 🔄 Smart shape recognition
- 📏 Ruler and guides for precise drawing
- 📁 Document-based architecture
- 🌙 Dark mode support

## Requirements
